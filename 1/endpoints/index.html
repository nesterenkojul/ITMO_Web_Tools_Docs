Эндпоинты

Поездка - Trip

  • просмотр списка поездок
    GET -> /trip/all
  • просмотр списка поездок текущего авторизированного пользователя
    GET -> /trip/my
  • просмотр одной поездки
    GET -> /trip/{trip_id}
  • создание новой поездки (текущий пользователь добавляется в неё автоматически)
    POST -> /trip
  • редактирование поездки (только в которой состоит текущий пользователь)
    PATCH -> /trip/edit/{trip_id}
  • удаление поездки (только в которой состоит текущий пользователь)
    DELETE -> /trip/delete/{trip_id}


Пользователь - User

  • просмотр списка пользователей
    GET -> /user/all
  • просмотр текущего авторизированного пользователя
    GET -> /user/me/pwd
  • изменение пароля текущего авторизированного пользователя
    PATCH -> /user/me/pwd
  • просмотр одного пользователя
    GET -> /user/{user_id}
  • создание (регистрация) нового пользователя
    POST -> /user/register
  • авторизация пользователя
    POST -> /user/login
  • редактирование пользователя
    PATCH -> /user/edit/{user_id}
  • удаление пользователя
    DELETE -> /user/delete/{user_id}


Связь Пользователь-Поездка - UserTripLink

  • просмотр списка связей
    GET -> /usertrip/all
  • просмотр одной связи
    GET -> /usertrip/{user_id}in{trip_id}
  • создание новой связи
    POST -> /usertrip
  • удаление связи (только в которой состоит текущий пользователь)
    DELETE -> /UserTripLink/delete/{user_id}in{trip_id}

Этап - Step

    GET -> /step/all

    GET -> /step/{step_id}
  • создание нового этапа (только поездки, в которой состоит текущий пользователь)

  • редактирование этапа (только поездки, в которой состоит текущий пользователь)
